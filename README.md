# 🌍 Project Overview

SafeRoute AI is an intelligent Disaster Management and Emergency Response Platform designed to help citizens, rescue teams, and authorities respond quickly during natural disasters and emergencies.

The platform provides:

* 🚨 Real-time SOS emergency alerts
* 🤖 AI-powered disaster assistance
* 🗺️ Live disaster monitoring dashboard
* 🚑 Rescue team coordination
* 📍 Shelter and safe-route guidance
* 📢 Emergency notifications
* 👥 Role-based access for citizens and administrators

# ⚙️ Installation

```bash
git clone https://github.com/Adeeb67/SafeRoute-AI.git

cd SafeRoute-AI

npm install
```

# 🔧 Environment Variables

Create a `.env` file:

```env
DATABASE_URL=
JWT_SECRET=
GEMINI_API_KEY=
```

# 🚀 Run Locally

```bash
docker-compose up --build
```

or

```bash
npm run dev
```
